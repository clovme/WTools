簡介

　  《APKDB 安卓逆手》（原名：Android APK+Dex檔案反組建及回組建工具）以下簡稱:APKDB 
　　APKDB是一款，針對Android OS系統用的APP及相關檔案，直接反組建修改的工具；
　　APKDB工具集合了當今最強悍，最犀利的與Android相關組建工具，如Apktool，Smali等等；
　　正常安裝后，它直接會在滑鼠右鍵創建【使用 APKDB 組建】快捷菜單；
　　本工具非常方便 AP K程式中文工作者，對APK或Dex等檔案進行簡易的重組建操作。
　　
___________________________________________________________________________________
功能特性

　· 對*.Apk 檔案 反組建、回組建、簽名、檢視簽名信息、極限壓縮等；
　· 對*.Dex 檔案 反組建、回組建、檢視詳細源代碼等；
　· 對*.odex 檔案，合并到apk或jar里面；
　· 對*.Apk、*.Zip 檔案 執行簽名、刷機等等；
　· 對*.jar 檔案 檢視源代碼、反組建；
　· 在Windows系統下直接顯示APK檔案自身圖示。

聲明：1.本工具必須先安裝JAVA8(JRE)以上環境；
　　　2.如需顯示APK自身圖示必須安裝.NET Framework4。

使用說明

    本工具執行安裝后，直接在滑鼠右鍵創建【使用 APKDB 組建】菜單。具體操作，如：
通過對APK或DEX檔案執行滑鼠右鍵選擇【使用 APKDB 組建】，在菜單里選擇相應的工具反組建，
會生成可同名資料夾，里面的檔案中文修改完成后，返回同名資料夾再次【使用 APKDB 組建】
即可，重新回組建成APK程式。
    對于JAR，OdeX以及ZIP也是同樣操作，具體會在【使用 APKDB 組建】菜單里顯示相關功能。
　　本工具可以在任意檔案(夾)打開菜單，里可以選擇最基本的操作，如：
設定默認Apktool版本，或者制作簽名包等等！
    針對多個APK檔案，可以放在同一個資料夾里，執行【使用 APKDB 組建】，會出現批量操作
菜單，可以執行全部順序反組建，修改好后，返回資料夾，可以同時回組建。無需逐個選擇。

常見問題

　　1、被殺毒軟件攔截誤殺（安裝前，關閉殺毒軟件）

　　2、UAC限制 安裝過程沒有寫入注冊表提示，直接顯示安裝完成
　　　（右鍵 我的電腦-管理-本地用戶和組-用戶-Administrator（√ 密碼永不過期）
　　　 請在系統變量【path】里最前面添加 c:\windows;c:\windows\system32;

　　3、安裝后無法組建檔案，提示JAVA 錯誤
　　  （未安裝JAVA平臺，或安裝錯誤。請卸載后重啟電腦，重裝再次重啟電腦）

　　4、安裝成功后，組建檔案提示 “Can't create the output directory \\****\smali”
　　  （這是資料夾 沒有寫入權限。如果不是，請檢查apk檔案名稱是不是有問題）

　　5、能反組建，但無法回組建
　　（可能是被舊版本的Apktool組建過，試試用舊版的反組建，用新版的回組建）

　　6、回組建出錯問題：
　　　　(1.根據提示 strings.xml 發現是最后一行錯誤，檢查是否</string>符號錯誤；
　　　　(2.根據提示 strings.xml 最頂部含中文代碼首行錯誤，編碼格式不對，轉換成 UTF-8；
　　　　(3.根據提示 public.xml 出錯，檢查改動過的 arrays.xml 是否代碼有錯誤的地方；
　　　　(4.其他錯誤，請到群里詢問相關熟悉的人員。

　　7、安裝過程提示 systeminfo  提示“不是內部或外部命令”
　　　　請在系統變量【path】里最前面添加 c:\windows;c:\windows\system32;
　　　　或者手動添加這個變量，如果還是不行可能系統不完整，重裝吧)

　　8、64位系統,如果JAVA出錯,建議把32位和64位的JAVA一起裝上。

　　9、其他基礎問題，請到QQ群(8023666)提問，不要單獨Q我，謝謝！

工具說明

1. APK檔案反組建工具 ApkTool 以及 ShakaApktool
　　用于直接反組建APK檔案，同時也會反組建dex檔案。
         ApkTool 官網：http://ibotpeaches.github.io/Apktool/
    ShakaApktool 官網：http://www.rover12421.com/

2. DEX檔案反組建工具 Baksmali/Smali
　　專門用來反組建Dex檔案，可以跟ApkTool兼容使用。
    官網：https://bitbucket.org/JesusFreke/smali/

3. 反組建資源檢視工具JADX
　　用來檢視反組建Dex后源碼的工具，據說能完整的反組建出Dex檔案源碼。
    官網：https://github.com/skylot/jadx

4. APK檔案簽名工具 SignApk
　　為APK檔案執行簽名，專門提供給廣大中文工作者們。

5. APK安裝器 刷機精靈PC版
　　為用戶提供可以預先檢視軟件的所有信息包括權限的APK安裝器。
    官網：http://www.shuame.com/

6. 顯示APK自身圖示 Apkshellext
　　該程式非常不錯，可惜占用緩存和讀取速度較慢，容易出現讀取滯留狀態，
　　從而導致APK檔案無法刪除，建議配置低的電腦慎用。
    官網：http://www.apkshellext.com/

7. Keytool簽名包制作工具 & 轉化工具MTkeytool
　　調用JAVA Keytool工具，非本工具自帶。
    轉化工具MTkeytool 用于轉化Keytool制作的簽名包，必須JAVA8
    官網：http://www.oracle.com/

8. OptiPNG圖片壓縮工具
    PNG 即 Portable Network Graphic 的簡稱，PNG 圖像是一種無損壓縮圖
    像檔案格式。因為網絡傳輸的需要，我們總是希望 PNG 圖像的容量能夠小
    些、小些、再小些。
    要優化 PNG 圖像，可以使用 Optipng 這個專門的 PNG 圖像優化工具。
    官網：http://optipng.sourceforge.net/
